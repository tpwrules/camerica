Preparing SD card:

* Download the LXDE image from Terasic and write it to the SD card
* Boot system
* Run expand_root_fs.sh on the desktop, then reboot
* Run resize2fs_once on the desktop, then reboot
* Delete both files

Now let's update the system

* apt-mark hold firefox # or it will crash forever
* apt update
* apt upgrade
* apt autoremove

Install misc useful packages

* apt install git 

Install python

* apt install python3 python3-setuptools python3-venv python3-dev
* pip3 install wheel

Install pygame

* apt install libsdl-dev libsdl-image1.2-dev libsdl-mixer1.2-dev libsdl-ttf2.0-dev 
* apt install libsmpeg-dev libportmidi-dev libavformat-dev libswscale-dev
* pip3 install pygame
